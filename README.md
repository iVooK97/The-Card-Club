# The Card Club - Next.js Starter

This is a real starter codebase for The Card Club.

## What is included

- Landing page
- App demo page
- Player profile
- Walkout screen
- Playable Shnarps demo logic
- Tournament/leaderboard/membership sections

## How to run

1. Install Node.js
2. Open Terminal in this folder
3. Run: npm install
4. Run: npm run dev
5. Open: http://localhost:3000
